# WAP WD Journal

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或具体网站，仅作为 HTML 页面的存储与展示空间。

## 项目简介

本项目的主要目的是提供一个结构化的方式来保存和发布独立的 HTML 页面。这些页面可能包含各种类型的内容，例如笔记、工具、演示页面或其他静态资源。所有页面均为纯前端实现，不依赖后端服务。

## 目录说明

```
/
├── index.html          # 仓库首页 / 页面导航
├── pages/              # 存放各独立 HTML 页面
│   ├── example-page-1/
│   │   └── index.html
│   └── example-page-2/
│       └── index.html
├── assets/             # 公共资源（CSS、JS、图片等）
└── README.md           # 本文件
```

- `index.html`：可作为页面入口或索引，方便浏览所有已归档页面。
- `pages/`：按目录存放独立页面，每个页面可包含自己的资源文件。
- `assets/`：存放多个页面间共享的样式、脚本或图片。

## 页面归档说明

- 每个独立页面应放置于 `pages/` 下的单独文件夹中，并确保 `index.html` 作为入口文件。
- 页面应尽量自包含，避免依赖外部资源（除非是公共资产）。
- 建议在页面内添加适当的元数据（如标题、描述、创建日期）以便后续维护。

## 维护说明

- 本仓库由维护者手动更新，不接受自动部署或持续集成。
- 新增页面时请遵循目录结构规范，并更新 `index.html` 中的导航链接（如有）。
- 若需修改或移除页面，请直接操作对应文件并提交 Pull Request。
- 本仓库不保证所有页面长期可用，维护者可能根据需要对内容进行调整。

## 贡献

欢迎通过 Issue 或 Pull Request 提出建议或提交新页面。提交前请确保页面内容不违反任何法律法规，且不包含恶意代码。

## 许可

本仓库中的内容遵循相应许可协议，具体请参考各页面内的说明。若无特殊声明，默认保留所有权利。
